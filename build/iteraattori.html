Opastus

Generoi pinolle sisältö painamalla Aloita-painiketta. Ota pinosta seuraava alkio painamalla Seuraava-painiketta. Aloita alusta painamalla Aloita-painiketta. Seuraava-painike lakkaa toimimasta kun iteraattori pääsee loppuun.

Lähdekoodi

Alla on esillä tietorakenteen kannalta oleellinen lähdekoodi TypeScriptillä kirjoitettuna

      class Iterator {
        pino: Stack
        current?: ListItem
      
        constructor(pino: Stack) {
          this.pino = pino
          this.current = pino.top
        }
      
        hasNext = (): boolean => {
          return this.current !== undefined
        }
      
        next = (): ListItem | undefined => {
          if (this.current) {
            const oldCurrent = this.current
            this.current = this.current.next
            return oldCurrent
          } else {
            return undefined
          }
        }
      }
